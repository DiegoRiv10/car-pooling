# Historias de Usuario

> *Documento pendiente de completar en sesiones posteriores.*

## Formato

**Como** [tipo de usuario], **quiero** [acción], **para** [beneficio].

---

### HU-001: Registro de usuario
**Como** estudiante universitario, **quiero** registrarme con mi correo institucional, **para** verificar que soy parte de la comunidad universitaria.

**Criterios de aceptación:**
- El sistema solo acepta correos con dominio institucional.
- Se envía un correo de verificación.
- El usuario no puede publicar ni solicitar viajes hasta verificar su cuenta.
